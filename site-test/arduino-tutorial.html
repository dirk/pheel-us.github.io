Arduino

Arduino is an open-source electronics prototyping platform based on flexible, easy-to-use hardware and software. It's intended for artists, designers, hobbyists and anyone interested in creating interactive objects or environments.

How do I get started?

  1. Begin by dowloading the free Arduino software here. Arduino's software is very similar to Processing and you can build upon what you learn.
  2. Purchase the components you want to work with. The sky is the limit here. I recommend beginning with a starter kit, which comes with an Arduino and some of the other things you'll need to get it running. Adafruit and Sparkfun both sell these components and others, and also offer excellent tutorials.
  3. Arduino is incredibly versatile and there is a lot of good documentation out there. I would recommend beginning with the tutorials offered by Adafruit and Sparkfun. Start with Lesson One, and move to Lesson Two, and so on.
